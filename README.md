前额叶实验室 (Prefrontal Lab) - 脑科学认知训练中心

这是一个基于 React 开发的单页面应用 (SPA)，集成了四项经典的神经科学认知训练任务。旨在通过科学设计的交互游戏，提升用户的前额叶皮层活跃度、工作记忆容量以及执行抑制功能。

🧠 核心训练模块

1. 舒尔特方格 (Schulte Table)

原理：视觉搜索广度与注意力分配训练。

目标：以最快速度按顺序点击 1-25 的数字。

认知提升：扩大视野范围，提高注意力的稳定性和视觉处理速度。

2. Stroop 干扰测试 (Stroop Effect)

原理：基于执行功能的抑制控制理论。

目标：忽略文字本身的意义，识别并点击文字呈现的物理颜色。

认知提升：强化冲突控制能力，减少冲动反应。

3. N-Back (2-Back)

原理：工作记忆 (Working Memory) 刷新任务。

目标：判断当前数字是否与“前前个”数字相同。

认知提升：这是目前脑科学界公认最有效的流体智力训练之一，能显著增加大脑处理复杂信息的能力。

4. SET 模式匹配 (Pattern Matching)

原理：逻辑识别与认知灵活性。

目标：在 9 张卡牌中快速识别具有相同颜色或形状属性的组合。

认知提升：增强分类思维与逻辑推理速度。

🚀 快速开始

该项目是一个纯前端应用，无需安装任何环境。

下载文件：获取 BrainTrain.html。

重命名：将其重命名为 index.html 以便适配 Web 服务器。

运行：双击在浏览器中打开，或部署至 GitHub Pages。

🛠️ 技术栈

框架: React 18

样式: Tailwind CSS

图标: Lucide Icons

部署: 兼容 GitHub Pages、Vercel 或本地 HTML 运行

📈 部署到 GitHub Pages

在 GitHub 上创建一个新的仓库。

上传 index.html 和本 README.md 文件。

进入仓库设置 (Settings) -> Pages。

选择 main 分支并点击 Save。

几分钟后，你的专属“脑科学实验室”就会上线！
